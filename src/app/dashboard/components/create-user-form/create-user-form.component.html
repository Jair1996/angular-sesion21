Cuando se encuentra en esta ruta y quiere navegar a otra aparecerá un alert que preguntará si queremos abandonar esta página, este alert solo se ejecutará si solo sí el usuario ha manipulado algún input es decir el createForm.pristine = false (dando a entender que ha querido registrar o está queriendo registrar algún usuario); si el usuario no ha manipulado ningún campo, ha presionado el botón "Crear Usuario" no aparecerá el alert.

En la siguiente línea verá como el pristine del formulario cambia dependiendo si algún campo del formulario ha sido tocado:

createForm.pristine = {{ createForm.pristine }}
